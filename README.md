# Black Swan Research

> Restore consciousness as recognized human capacity. Recover suppressed research. Build the open replication archive Anglo science abandoned after the 1960s.

This is the public research library of the [Black Swan Project](https://blackswanp.com) — a bilingual ES/EN, open-source archive of consciousness research, citations, replications, and structured analysis.

We publish the work. You can cite it, fork it, translate it, replicate the experiments, and submit corroborating sources via pull request.

---

## What's in this repo

| Folder | What |
|---|---|
| [`articles/`](./articles) | Long-form research articles (bilingual ES + EN). Each article has its primary sources, evidence-tier markup, and citation chain. |
| [`pillars/`](./pillars) | Structured entity database — scientists, institutions, phenomena, government programs, practitioners, ancient wisdom lineages. |
| [`lineages/`](./lineages) | Practice traditions (Theravada, Vajrayana, etc.) and their documented effects. |
| [`best/`](./best) | Evidence-based protocol guides. |
| [`compare/`](./compare) | Side-by-side analyses of techniques, compounds, frameworks. |
| [`lead-magnets/`](./lead-magnets) | Compiled reading lists and curated source material. |
| [`press/`](./press) | Verified third-party coverage. |
| [`glossary.ts`](./glossary.ts) | Public glossary of consciousness-research terms. |

## What's NOT in this repo

This is the research library, not the company. We deliberately exclude:

- Course content (paid products live on blackswanp.com)
- Marketing strategy, ad copy, funnel mechanics, financial data
- Email sequences, subscriber data, partner relationships
- The Next.js application code that powers the public site

If you want the polished, designed reading experience, visit [blackswanp.com](https://blackswanp.com). If you want the underlying data, source chains, and evidence rubric — you're in the right place.

## Evidence rubric

Every claim in this repo is marked with one of four tiers (see [EVIDENCE-RUBRIC.md](./EVIDENCE-RUBRIC.md) for the full definition):

| Tier | Meaning |
|---|---|
| `CORROBORATED` | Multiple independent peer-reviewed sources confirm the claim. |
| `CONTESTED` | Sources disagree; we present both sides. |
| `OPEN-CASE` | Insufficient evidence to settle; an active question. |
| `SPECULATIVE` | A working hypothesis, framed as such. |

In MDX articles this appears as: `<AnswerCapsule evidence="CORROBORATED" source="...">`. Preserve this markup when editing — it's load-bearing.

## How to contribute

We welcome:

1. **Citation fixes** — broken links, wrong dates, missing primary sources.
2. **Corroborating sources** — strengthen an existing claim with another peer-reviewed study.
3. **Replication reports** — you tried a protocol (Wim Hof, Gateway Process, breath retention, magnesium stack, manifestation practice), here's your n=1 or n-of-many data.
4. **Translation polish** — improvements to ES↔EN parallels.
5. **New article proposals** — open an issue first; we'll triage before you write.

See [CONTRIBUTING.md](./CONTRIBUTING.md) for the full process and the PR checklist. See [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) for community standards.
