# Black Swan Research

> The science of consciousness was abandoned by Anglo institutions after the 1960s. Latin American researchers never stopped. We pick it back up bilingually, in the open, with primary sources, and with the discipline to mark what we know vs what we suspect.

This is the public research vault of the [Black Swan Project](https://blackswanp.com) — a bilingual ES/EN archive of consciousness research, citations, protocols, and replication reports. Released under [CC BY-SA 4.0](./LICENSE).

You can read it, cite it, fork it, translate it, replicate the experiments, and submit findings via pull request.

## How to navigate

**Browse by entry point:**

| If you want… | Start here |
|---|---|
| To read about a topic or researcher | [`research/`](./research) — articles organized by pillar (scientists, government programs, phenomena, practitioners, ancient wisdom, institutions, mental work) |
| Step-by-step protocols you can run | [`protocols/`](./protocols) — practical guides and comparisons (cold plunge, magnesium, breathwork, vagus nerve) |
| Curated source bibliographies | [`sources/`](./sources) — compiled reading lists and primary-source collections |
| To submit your own replication report | [`replications/`](./replications) — n=1 / n-of-many results from contributors |
| Reference data (taxonomies, press, glossary) | [`reference/`](./reference) — the structured database the archive is built on |
| The full list of every article | [`INDEX.md`](./INDEX.md) |

**Browse by pillar:**

- [`research/scientists/`](./research/scientists) — researchers documenting consciousness phenomena (Grinberg, Hoffman, Kastrup, Sheldrake, Tononi)
- [`research/institutions/`](./research/institutions) — labs, universities, research programs
- [`research/government-programs/`](./research/government-programs) — Stargate, MKUltra, Soviet psi, Gateway Process, declassified state research
- [`research/phenomena/`](./research/phenomena) — psi, ganzfeld, global consciousness, anomalous cognition
- [`research/practitioners/`](./research/practitioners) — Wim Hof, Joe Dispenza, Grof, manifestation methodologists
- [`research/ancient-wisdom/`](./research/ancient-wisdom) — documented traditions and lineages
- [`research/mental-work/`](./research/mental-work) — sleep, attention, anxiety, focus interventions

## Article format

Every article folder contains:

```
research/<pillar>/<slug>/
├── en.mdx        ← English long-form
├── es.mdx        ← Spanish long-form
├── meta.json     ← title, sources, dates, evidence tier, pillar
└── social.json   ← short-form summaries (where present)
```

Every substantive claim is marked with an **evidence tier** — `CORROBORATED`, `CONTESTED`, `OPEN-CASE`, or `SPECULATIVE`. See [EVIDENCE-RUBRIC.md](./EVIDENCE-RUBRIC.md) for definitions. In MDX this appears as `<AnswerCapsule evidence="CORROBORATED" source="…">` — preserve that markup when editing; it's load-bearing.

## How to contribute

We welcome:

1. **Citation fixes** — broken links, wrong dates, missing primary sources
2. **Corroborating sources** — strengthen an existing claim with another peer-reviewed study
3. **Replication reports** — you ran a protocol, here's your data → drop in [`replications/`](./replications)
4. **Translation polish** — improvements to ES↔EN parallels
5. **New article proposals** — open an issue first; we'll triage before you write

See [CONTRIBUTING.md](./CONTRIBUTING.md) for the full process. See [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) for community standards.

By contributing you agree your work is licensed under [CC BY-SA 4.0](./LICENSE).

## What's NOT in this vault

This is the research library, not the company. We deliberately exclude:

- Paid course content (lives on blackswanp.com)
- Marketing strategy, ad copy, funnel mechanics, financial data
- Email sequences, subscriber data, partner relationships
- The Next.js application code that powers the public site

If you want the polished reading experience, visit [blackswanp.com](https://blackswanp.com). If you want the underlying data and source chains — you're already there.
