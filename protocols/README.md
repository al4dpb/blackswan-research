# Protocols

Practical, evidence-graded protocols you can replicate. Each folder is a single technique or comparison: how to run it, what the data says, what would invalidate it.

Two formats live here:
- **Guides** (`<technique>-guide/`) — single-protocol walkthroughs with sourced effect sizes.
- **Comparisons** (`<a>-vs-<b>/`) — side-by-side analysis of competing approaches.

If you replicate one of these protocols, drop your data in [../replications/](../replications/) — that's the citable record we build over time.
